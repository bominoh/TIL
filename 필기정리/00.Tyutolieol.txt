

- ls : 현재 위치 폴더/파일 
- cd .. : 뒤로 나가기
- cd <폴더명>: 폴더로 이동
- mkdir <폴더명> : 폴더생성
- touch <파일명><확장자> : 파일생성
- vi <파일명><확장자> : 수정
  => 또한, 새파일 생성/수정 가능
- rm -r <폴더명> : 폴더삭제
- rm <파일명> : 파일삭제
- mv <폴더명><확장자> <옮길 파일명> : 폴더->파일로 이동
- mv <파일명><확장자> <변경파일명><확장자> : 파일이름 변경
- cp <파일명><확장자> <복사 파일명><확장자> : 파일복사

- clear : git bash 내용모두 삭제
- Tab : 자동완성 단축키
